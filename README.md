# TanStack Query with React

This repo contains a React app that shows different ways to fetch and manage API data. It includes examples with React and with TanStack Query features.

## What this project includes
- Fetching data with useEffect
- Manual loading, error, and state handling
- Basic useQuery for fetching data
- useQueries for running multiple API requests
- useMutation for creating data and updating the cache
- Pagination using query keys with page numbers
- Config options like staleTime and refetchInterval

## API
All examples use the JSONPlaceholder test API.
